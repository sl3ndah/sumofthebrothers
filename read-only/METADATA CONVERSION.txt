## 🎼 Smart Playlists

**New tab structure:**
```
[All] [Joey] [Sum] [Smart Playlists ▼]
                    └─ Uploaded This Week
                    └─ Chill Vibes
                    └─ Latest Versions Only
                    └─ Electronic Tracks
                    └─ [+ Create Smart Playlist]
```

### Smart Playlist Editor:
```
┌─────────────────────────────────────────┐
│  Create Smart Playlist              [×] │
├─────────────────────────────────────────┤
│                                         │
│  Name: [Chill Electronic___________]    │
│                                         │
│  Match: [All] [Any] of these rules:     │
│                                         │
│  [Genre] [contains] [electronic___] [×] │
│  [Mood]  [contains] [chill________] [×] │
│  [BPM]   [between]  [80] and [120 ] [×] │
│                                         │
│  [+ Add Rule]                           │
│                                         │
│  Sort by: [Uploaded Date ▼] [Desc ▼]   │
│  Limit:   [No limit ▼]                  │
│                                         │
│  [Save] [Cancel]                        │
└─────────────────────────────────────────┘
```

### Available Filters:
```
Time-based:
- Uploaded: today, this week, this month, this year, before/after date
- Created: today, this week, this month, this year, before/after date
- Released: today, this week, this month, this year, before/after date

Metadata:
- Artist: is, is not, contains
- Genre: contains, does not contain, is empty
- Mood: contains, does not contain, is empty
- BPM: equals, between, greater than, less than
- Key: is, is not, is empty
- Duration: greater than, less than, between

Version filters:
- Version: first, last, specific number, all versions
- Only latest version per track

Other:
- Playlist: is, is not
- Has lyrics: yes, no
- Has notes: yes, no
```

## ✅ Pros

### Track Info Modal:
- ✅ Clean UI - no clutter
- ✅ All metadata in one place
- ✅ Easy to read lyrics/notes
- ✅ Can add "Edit" button for quick edits
- ✅ Shareable (can copy link to specific track info)

### Flatten Track Structure:
- ✅ Simpler navigation (2 levels vs 3)
- ✅ Faster to browse
- ✅ Version badges show at a glance
- ✅ Less clicking to find tracks

### Smart Playlists:
- ✅ Powerful discovery ("show me all chill tracks")
- ✅ Dynamic (auto-updates when you add tracks)
- ✅ Multiple use cases (by mood, genre, date, etc)
- ✅ Can create "New This Week" that auto-updates
- ✅ Version filtering (show only latest versions)

---

## ⚠️ Cons / Challenges

### Track Info Modal:
- ⚠️ Extra click to see info
- ⚠️ Need good mobile layout
- ⚠️ Could be slow if lots of data

**Solutions:**
- Keyboard shortcut (press 'i' for info)
- Responsive modal design
- Lazy load lyrics/notes

### Flatten Structure:
- ⚠️ All versions shown = long lists
- ⚠️ Harder to see "all versions of floataway together"

**Solutions:**
- **Version grouping badge**: Show "floataway (3 versions)" with expand
- **Smart playlist**: "Latest Versions Only" by default
- **Filter toggle**: "Show all versions" checkbox

### Smart Playlists:
- ⚠️ Complex to build UI
- ⚠️ Need to store playlist definitions
- ⚠️ Performance with many filters

**Solutions:**
- Start with simple presets
- Store in localStorage
- Cache filtered results
